声の高さ チェック

ここでは基本周波数を用います。
周波数が高いほど、高い声、逆に、周波数が低いと低い声になります。

基本周波数: 0 [Hz]

戻る ホーム画面へ
- numpy - librosa import numpy as np import librosa #音声データを受け取り、librosaを使って基本周波数を計算 def process_audio(audio_data): audio = np.frombuffer(audio_data, dtype=np.int16).astype(np.float64) fo = librosa.yin(audio, fmin=50, fmax=500, sr=44100) fo_mean = np.mean(fo) if len(fo) > 0 else 0 return fo_mean #計算された基本周波数をHTMLのoutput要素に表示 def update_output(fo_mean): document.getElementById('output').innerText = f"基本周波数: {fo_mean:.1f} [Hz]"
